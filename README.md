# Cellbit VODs

"Cellbit VODS" é um projeto baseado nos site de vods do [XQC](https://xqc.wtf) e [moonmoon](https://moon2.tv/). O projeto está sendo feito com [Next.js](https://nextjs.org).

> [!IMPORTANT]
> O projeto **não** é OFICIAL e não tem quaisquer envolvimento com o Cellbit ou com Ordem Paranormal.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
